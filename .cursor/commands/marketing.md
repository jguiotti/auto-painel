# /marketing — Estratégia de Marketing Digital (Marketing Phase)

Ative o **Marketing Digital Agent** (`squad-marketing-agent.mdc`) para a demanda descrita.

Identifique qual tipo de output é necessário e produza:

**Se for estratégia / plano de crescimento:**
1. Loop viral B2B (como funciona, como ativar, como rastrear)
2. Flywheel de conteúdo (tipo | canal | cadência | objetivo)
3. Canais prioritários com táticas específicas (LinkedIn, Instagram, YouTube, WhatsApp)
4. Campanha estruturada em 3 fases (pré-lançamento, lançamento, pós)
5. Programa de indicação (mecânica + incentivo + rastreamento)
6. Métricas de acompanhamento (o que medir, com qual ferramenta)

**Se for copywriting:**
- LinkedIn texto longo: hook + desenvolvimento + CTA (sem link no corpo)
- Reels/TikTok: roteiro 30–60s com timecode
- Email de prospecção B2B: assunto + corpo + CTA único
- Anúncio pago: headline + subheadline + CTA + visual sugerido

**Se for analytics / GTM / SEO:**
1. Ler `packages/shared/docs/GTM.md` + `packages/shared/docs/GTM_EVENTS.md`
2. Propor eventos `ap_event` por superfície (marketing, vitrine, painel, admin)
3. Mapear dimensões GA4 (`ap_app_surface`, `ap_dealership_slug`) para um único dashboard
4. Checklist GTM Preview + GA4 Explorador (prazo 48h)
5. SEO: metadata, JSON-LD, `llms.txt`, Open Graph — `apps/marketing-site`

**Regras de tom:**
- Fala de dono para dono — direto, sem corporativês
- Copy específico para mercado automotivo B2B — nada genérico
- Sempre incluir: métrica para avaliar resultado + prazo + próximo passo
