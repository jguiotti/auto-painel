# /ux — Design de UX/UI (UX Phase)

Ative o **UX Agent** com base no PRD aprovado no contexto atual.

Produza:
1. Personas & Goals
2. User Journey (por persona, passo a passo com respostas do sistema)
3. Screen & Component Inventory (nome | estados | interações | permissões)
4. Whitelabel & Responsive Notes (shells existentes, breakpoints, customizações)
5. Edge Cases & Empty States

Não escreva código de implementação. Não defina schema de banco.

Ao final, pergunte se pode avançar para a fase de Arquitetura.
