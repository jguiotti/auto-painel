# /ux — Design UX/UI (UX Phase)

Ative o UX Agent com base no PRD e copy aprovados.

Produza:
1. Affected Apps & Personas (por app)
2. User Journey por persona (passo a passo com respostas do sistema)
3. Screen & Component Inventory (component | app | estados | interações | permissões | @autopainel/shared/ui ou novo?)
4. Design System Mapping (primitivas existentes vs. componentes novos para packages/shared)
5. Whitelabel & Tenant Customization
6. Responsive Notes (mobile/tablet/desktop por app)
7. Edge Cases com copy do UX Writer (não inventar copy)

Não escreva código. Flag componentes novos para packages/shared.
Ao final, perguntar se passa para Arquiteto Supabase.
